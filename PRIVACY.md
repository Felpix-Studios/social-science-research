# Privacy Policy

This plugin is distributed as a local Codex plugin and is intended to operate on files in the user's workspace.

## Data Handling

- The plugin does not include a hosted backend service.
- Skills, hooks, and scripts operate on local project files unless the user or runtime explicitly invokes external tools or web access.
- Any network access is initiated by the Codex runtime, the user, or tools used by individual skills.

## User Content

- Research notes, code, manuscripts, and data remain in the user's local environment unless the user chooses to send them elsewhere.
- Users are responsible for reviewing individual skills before using them with sensitive or regulated data.
